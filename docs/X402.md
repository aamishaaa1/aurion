# x402 Micropayment Protocol

## Overview

x402 enables micropayments for AI verification services.

## Payment Flow

```
User Request → Create Session → Record Usage → Calculate Cost → Payment → Settlement
```

## Usage Pricing

- Text verification: 0.001 NEURO
- Image verification: 0.005 NEURO
- Video verification: 0.010 NEURO
- DKG publish: 0.002 NEURO

## Implementation

```typescript
import { x402Gateway } from '@aurion/trust';

// Create session
const sessionId = await x402Gateway.createSession(userId);

// Record usage
await x402Gateway.recordUsage(sessionId, 'VERIFY_TEXT', 0.001);

// Initiate payment
const payment = await x402Gateway.initiatePayment(
  sessionId,
  senderAddress,
  recipientAddress
);

// Get session
const session = await x402Gateway.getSession(sessionId);
```

## Settlement

```typescript
import { settlementEngine } from '@aurion/trust';

const settlement = await settlementEngine.createSettlement(sessionId);
```

## Currency

Payments are denominated in NEURO tokens on NeuroWeb.
